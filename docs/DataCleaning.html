Labels modified

In the original data set the “Label” column has three categories i.e. ‘<5’, ‘>30’ and ‘NO’

We have lumped the factors ‘<5’ and ‘>30’ in a new category names ‘YES’.

So now the “Label” column has two categories i.e. ‘YES’ and ‘NO’
